
Senior-style Python Playwright framework using:
- Pytest
- Playwright
- API setup for stable test data
- storage state for authenticated UI flows
- Allure reporting
- `.env` configuration
- GitHub Actions CI

## Why this repo looks strong on GitHub

Instead of creating todos only through the UI, this framework creates test data through API and validates it through UI.

That shows:
- faster test execution
- cleaner test design
- less flaky setup
- stronger automation architecture
